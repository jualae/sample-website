# 마누스 샘플 웹사이트

GitHub Pages로 배포된 샘플 웹사이트입니다.

## 개요

이 프로젝트는 마누스 AI 에이전트가 생성한 샘플 웹사이트입니다. HTML, CSS, JavaScript를 사용하여 만들어졌으며, GitHub Pages를 통해 무료로 배포됩니다.

## 특징

- ✨ **반응형 디자인** - 모바일, 태블릿, 데스크톱 모두에서 완벽하게 작동합니다.
- 🚀 **빠른 로딩** - 최소한의 리소스로 빠른 로딩 속도를 제공합니다.
- 🎨 **깔끔한 UI** - 현대적이고 사용하기 쉬운 인터페이스입니다.
- 🔒 **안전** - GitHub Pages에서 자동으로 HTTPS를 제공합니다.
- 💰 **무료** - 배포 비용이 전혀 들지 않습니다.

## 파일 구조

```
sample-website/
├── index.html      # 메인 HTML 파일
├── styles.css      # 스타일시트
├── script.js       # JavaScript 파일
└── README.md       # 이 파일
```

## 사용 방법

1. 이 저장소를 클론합니다.
2. 로컬에서 `index.html`을 브라우저로 열어 확인합니다.
3. 필요에 따라 HTML, CSS, JavaScript를 수정합니다.
4. 변경사항을 GitHub에 푸시하면 자동으로 배포됩니다.

## 커스터마이징

### 제목 변경
`index.html`의 `<title>` 태그를 수정하세요.

### 색상 변경
`styles.css`의 색상 값을 수정하세요. 주요 색상:
- 주 색상: `#667eea`
- 보조 색상: `#764ba2`
- 배경색: `#f4f4f4`

### 콘텐츠 수정
`index.html`의 텍스트와 섹션을 수정하여 원하는 내용으로 변경하세요.
